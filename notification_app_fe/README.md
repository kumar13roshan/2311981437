# Notification Frontend

React + Material UI frontend for notification management and priority inbox.

## Run

```bash
npm install
cp .env.example .env
npm run dev
```

The app runs only on `http://localhost:3000` using Vite `--strictPort`.

## Features

- Responsive desktop and mobile UI
- Notification cards
- Filters by type
- Pagination / load more
- Priority inbox
- Loading skeletons
- Error and retry states
- Frontend logging for lifecycle, API calls, filters, and user actions
